Generator for Umalla script

Input

Only letters and full stops appear in the script. Thus, the example input is reduced to "shallicomparetheetoasummersday". You can add full stops between messages if you want to.

Thickness

Only even numbers produce nice results.

How to save

You can save the image by right-clicking on it. After that, you can edit it a program like Photoshop or Inkscape.
